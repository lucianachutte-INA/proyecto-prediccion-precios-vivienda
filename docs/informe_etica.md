# Informe breve de ética

## Proyecto: Predicción de precios de viviendas

El presente proyecto utiliza datos sintéticos generados con fines exclusivamente académicos. Esto significa que no se utilizaron datos personales, sensibles ni pertenecientes a personas reales.

Desde una perspectiva ética, el uso de modelos predictivos en el mercado inmobiliario debe realizarse con responsabilidad, ya que una predicción incorrecta podría influir en decisiones económicas importantes. Por este motivo, es fundamental aclarar que el modelo desarrollado no reemplaza una tasación profesional ni contempla todos los factores reales que intervienen en el valor de una vivienda.

También es importante considerar que, en un proyecto real, los datos utilizados podrían contener sesgos relacionados con la ubicación geográfica, el nivel socioeconómico, el acceso a servicios o las características del entorno. Si estos sesgos no son identificados y tratados correctamente, el modelo podría reproducir desigualdades existentes.

Para un uso responsable de este tipo de soluciones, se recomienda:

* Utilizar fuentes de datos confiables y actualizadas.
* Evitar el uso de información personal innecesaria.
* Evaluar posibles sesgos en los datos.
* Comunicar claramente las limitaciones del modelo.
* Utilizar las predicciones como apoyo a la toma de decisiones y no como única fuente de verdad.

## Conclusión

Este proyecto respeta principios éticos básicos al trabajar con datos simulados, documentar sus limitaciones y presentar sus resultados con responsabilidad. Su finalidad es educativa y busca demostrar el proceso de desarrollo de una solución de Ciencia de Datos en un contexto profesional simulado.
